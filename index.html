Perguntas Frequentes

Posso usar esse template no meu portfólio?
Sim — sinta-se livre para adaptar. Remova o crédito no rodapé se for entregar como trabalho individual.
Como executo o preview localmente?
Basta abrir o arquivo `index.html` no seu navegador ou usar uma extensão de live-server no VS Code.
O formulário envia dados?
Não nesta versão — o formulário é apenas para demonstração. Para envio real adicione um `script.js` com fetch/POST.
