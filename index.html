Sobre o projeto

Projeto criado para demonstrar uso de propriedades CSS como flexbox, grid, box-shadow, transitions, , pseudo-elements e técnicas de responsividade com @media.

Layout responsivo

Uso de unidades relativas, breakpoints e comportamento adaptativo para telas pequenas.

Interatividade

Transições, hover effects e foco para tornar a navegação mais agradável.

Acessibilidade

Contraste de cores, estados de foco e atributos ARIA quando aplicável.
